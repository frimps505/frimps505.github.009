HTML, CSS & JavaScript Cheat Sheet

A personal reference guide for commonly used HTML elements, CSS properties, and JavaScript functionalities used in my web development projects.

HTML Cheat Sheet

  • <!doctype html> – Declares HTML5 document type
  • <html lang="en"> – Sets document language
  • <head> – Contains metadata and links
  • <title> – Page title in browser tab
  • <link rel="stylesheet"> – Links CSS file
  • <header> – Page header content
  • <nav> – Navigation menu
  • <section> – Groups related content
  • <div> – General container
  • <a href=""> – Hyperlink
  • <img src="" alt=""> – Displays images
  • class – Used for CSS styling
  • id – Unique identifier for JS/CSS
  • aria-label – Improves accessibility

CSS Cheat Sheet

  • color – Changes text color
  • background-color – Sets background color
  • font-family – Changes font style
  • font-size – Controls text size
  • margin – Space outside an element
  • padding – Space inside an element
  • border – Adds border around elements
  • display – Controls layout behavior
  • flex – Enables flexible layouts
  • justify-content – Aligns items horizontally
  • align-items – Aligns items vertically
  • width / height – Sets element size
  • hover – Style on mouse hover

JavaScript Cheat Sheet

  • const – Declares variables that cannot be reassigned
  • document.querySelectorAll() – Selects all elements with a specific class
  • document.querySelector() – Selects the first matching element
  • document.getElementById() – Selects an element by its ID
  • forEach() – Loops through selected elements
  • addEventListener() – Listens for user interactions
  • mouseenter – Triggered when the mouse enters an element
  • mouseleave – Triggered when the mouse leaves an element
  • animationend – Detects when a CSS animation finishes
  • classList.add() – Adds a CSS class dynamically
  • classList.remove() – Removes a CSS class dynamically
  • style – Modifies CSS styles using JavaScript
  • transition – Applies smooth animation effects
  • transform: scale() – Enlarges or shrinks elements
  • boxShadow – Adds visual depth using shadows
  • Arrow functions (=>) – Short syntax for functions
  • Event-driven programming – Code runs based on user actions
  • DOM manipulation – JavaScript dynamically updates page behavior
